# POS- Point of Sale System
a lightweight, web-based point of sale and sales management system built using native HTML/CSS/JavaScript
frontend and a Django backend

## Tech Stack
-Frontend: HTML,CSS,JavaScript
-Backend: Django
-Database:MySQL
_Auth: Django session-based authentication


## Features
Inventory management
Order creation and processing
Customer management
Payment tracking
Sales reports
## Prerequisites
-Python 3.10+
-pip

## Installation
```bash
    git clone https://github.com/banaye/ppSales.git
cd poe
pip install -r requirements.txt

```
## Setup
- python manage.py migrate
- python manage.py createsuperuser
-python manage.py runserver
